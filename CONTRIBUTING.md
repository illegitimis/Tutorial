# Contributing

Thanks for your interest in contributing!

## How to Contribute

1. Fork the repository
2. Create a branch for your changes
3. Make your edits (Markdown files in the relevant category folder)
4. Commit with a clear message
5. Open a pull request with a description of your changes

## Content Guidelines

- Use Markdown format with kebab-case filenames
- Follow the heading casing and link conventions in [CLAUDE.md](./CLAUDE.md)
- Every folder must have an `index.md` hub page
- Every leaf file ends with a breadcrumb nav footer (depth-adjusted)
- Test locally with `jekyll serve` if possible

## Review

All pull requests are reviewed before merging. Please be patient!
