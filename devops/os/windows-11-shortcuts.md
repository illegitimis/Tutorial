# Windows 11 Shortcuts

## Virtual Desktop

| Key Combination | Description |
|---|---|
| Win+Ctrl+Right | Switch to the next desktop |
| Win+Ctrl+Left | Switch to the previous desktop |
| Win+Ctrl+Shift+Right | Move current window to the next desktop [1] |
| Win+Ctrl+Shift+Left | Move current window to the previous desktop [1] |
| Win+Tab | Open Task View |
| Win+Ctrl+D | Create a new Virtual Desktop |
| Arrow Keys + Enter | Select a Virtual Desktop in Task View |
| Delete | Remove the selected desktop (in Task View) |
| Escape | Close Task View |

## Window Management

| Key Combination | Description |
|---|---|
| Win+D | Show the desktop |
| Win+M | Minimize all windows |
| Win+Shift+M | Maximize all windows |
| Win+, (comma) | Peek at the desktop |
